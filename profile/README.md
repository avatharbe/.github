# Avathar

phpBB extensions and styles for gaming guild communities.

[Avathar.be Forum Community](https://www.avathar.be/forum)

## Projects

- [Kanban Board](https://github.com/orgs/avatharbe/projects/1) — issue tracking across all repositories

## Core Extensions

| Extension | Description |
|-----------|-------------|
| [bbGuild](https://github.com/avatharbe/bbguild) | Guild management for phpBB — roster, portal, multi-game support |
| [bbGuild WoW](https://github.com/avatharbe/bbguild_wow) | World of Warcraft game plugin with Battle.net API integration |
| [bbDKP](https://github.com/avatharbe/bbDKP) | DKP point system for raid loot distribution (in development) |

## Game Plugins

bbGuild supports multiple games through separate plugin extensions:

[WoW](https://github.com/avatharbe/bbguild_wow) | [SWTOR](https://github.com/avatharbe/bbguild_swtor) | [FFXIV](https://github.com/avatharbe/bbguild_ffxiv) | [FFXI](https://github.com/avatharbe/bbguild_ffxi) | [GW2](https://github.com/avatharbe/bbguild_gw2) | [LotRO](https://github.com/avatharbe/bbguild_lotro) | [EQ](https://github.com/avatharbe/bbguild_eq) | [EQ2](https://github.com/avatharbe/bbguild_eq2) | [Lineage 2](https://github.com/avatharbe/bbguild_lineage2)
